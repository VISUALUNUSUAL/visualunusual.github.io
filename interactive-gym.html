Interactive fitness

Moscow, Russia
Role: Creative Director / Lead Designer

This interactive fitness installation transforms a gym into a responsive media environment where physical activity becomes a visual experience.

Each exercise machine is equipped with sensors capturing real-time workout data — speed, repetitions, distance — and converting them into generative LED lighting effects and large-scale screen animations.

The goal was to motivate users through visual feedback and create an immersive sense of progress and achievement.

We designed an interactive environment where physical activity generates real-time visual feedback. Sensors on each machine streamed live workout data (speed, distance, repetitions), which triggered generative animations on an 8K screen and synchronized lighting behavior across more than 20,000 LEDs in 21 contours.

The entire system reacted instantly, highlighting achievements and personal bests to encourage engagement and friendly competition.

Tech stack:
Java (data processing), WebSocket (transport), MIDI (event triggers), Resolume (visual generation), ArtNet (LED control), Processing (prototyping and testing)

The gym became an immersive space where every workout turned into a shared visual event. Users trained longer, reacted to visualized progress, and competed for recognition on the big screen.
